# Roadmap del Proyecto

Esta es la ruta de desarrollo planificada para futuras iteraciones de **InversiónSegura**:

## Fase 1: MVP (Actual)
- [x] Búsqueda de Tickers mediante Alpha Vantage.
- [x] Desglose de estado de resultados (Ingresos, EBITDA, Net Income).
- [x] Visualización de datos mediante gráficos interactivos.
- [x] Autenticación y base de datos de usuarios (Supabase).
- [x] Documentación técnica y arquitectura completa.

## Fase 2: Expansión de Análisis
- [ ] Implementar la consulta de la hoja de balance (Balance Sheet) y Flujo de Caja (Cash Flow).
- [ ] Incorporar cálculo automático de ratios financieros (Liquidez, Solvencia).
- [ ] Añadir exportación de datos en formato PDF y CSV.

## Fase 3: Funciones Avanzadas de Usuario
- [ ] Implementar sistema de seguimiento de "Favoritos" en el portafolio del usuario.
- [ ] Alertas por correo electrónico ante cambios bruscos en métricas fundamentales.
- [ ] Sistema de notificaciones en tiempo real (WebSockets).

## Fase 4: Optimización y Producción
- [ ] Despliegue en entornos Serverless (Frontend) y PaaS (Backend).
- [ ] Migración completa de archivos de caché en memoria a Redis.
- [ ] Internacionalización (i18n) para soportar múltiples idiomas (Español / Inglés).
